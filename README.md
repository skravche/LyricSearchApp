# Search Lyric App

> App that finds song lyrics using React and the Musixmatch API.

## Musixmatch API

Get your API here:

[https://developer.musixmatch.com](https://developer.musixmatch.com)

## Quick Start

bash

# Install dependencies

npm install

# Serve on localhost:3000

npm start

# Build for production

npm run build
